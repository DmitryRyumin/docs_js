# NodeJS

## [Установка Node](https://nodejs.org/en/)

### Узнать версию None

```shell script
node -v
# Или
node --version
```

### Тестирование Node

1. Создать файл `hello.js`

    ```javascript
    console.log('Привет из Node!');
    ```

2. Запустить через `Node`

    ```shell script
    node hello.js
    ```
